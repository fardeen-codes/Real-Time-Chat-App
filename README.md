# 💬 Real Time Chat Application

A **Real Time Chat Application** built using the **MERN Stack** that allows users to communicate instantly through messages. The application supports real-time communication using **WebSockets** and provides a smooth and responsive user experience.

## 🚀 Features

* 🔐 User Authentication (Signup & Login)
* 💬 Real-time messaging
* 👥 One-to-one chat
* 🟢 Online/Offline user status
* 📱 Responsive UI
* 🔔 Instant message updates using sockets

## 🛠 Tech Stack

**Frontend**

* React
* JavaScript
* HTML
* CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Real-time Communication**

* Socket.io

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/fardeen-codes/real-time-chat-app.git
```

### 2️⃣ Install dependencies

Backend

```
cd backend
npm install
```

Frontend

```
cd frontend
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file inside the **backend** folder.

Example:

```
PORT=
MONGO_URI=
JWT_SECRET_KEY=
```

### 4️⃣ Run the application

Backend

```
npm run dev
```

Frontend

```
npm run start
```
